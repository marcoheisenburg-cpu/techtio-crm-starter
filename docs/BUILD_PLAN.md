# Techtio Ops CRM Build Plan

## MVP Goal
Create a strong internal CRM for daily media buying operations. The first version tracks spend, accounts, agencies, buyers, offers, geos, leads, FTDs, revenue, profit and ROI.

## Phase 1
- Dashboard
- Shared media buying pool budget
- Remaining pool budget per buyer
- Daily spend manual entry
- Accounts table
- Agencies table
- Offers table
- Buyers table
- Geo and agency reports
- Supabase database schema

## Phase 2
- Supabase Auth login
- Real database reads/writes
- CSV imports
- Telegram daily reports
- Keitaro lead/click import

## Phase 3
- Meta Ads spend import
- Trackbox lead/FTD sync
- Alert rules
- Agency quality score
- Account ban tracking
- Buyer performance scoring
- Budget pacing recommendations

## Main KPIs
- Spend
- Leads
- CPL
- FTDs
- CPA
- Revenue
- Profit
- ROI
- Pool budget used
- Remaining pool budget
- Buyer budget remaining
- Account survival rate
- Agency quality score
